# Helmat: начало 2024
Учебный проект: сложная игра на processing без движка.
У этой игры есть:
- стартовый экран
- экран настроек с настройкой уровня громкости музыки и эффектов, а также с разными расширениями экрана (у игры есть адаптив)
- кат-сцены
- первый уровень, где игроку предстоит ответить на математический вопрос, найдя нужное число на карте пляжа (фейковые ответы там также разбросаны). За неправильные ответы теряется жизни. Всего их 3.
- второй уровень: платформер. Игрок управляет треугольником и пытается добраться до портала, после которого его ждёт...
- ...третий уровень: нужно ответить на математический вопрос. Неправильный ответ также ведёт к потери жизни
- Если игрок успешно всё прошёл, высвечивается хорошая концовка, а если нет - плохая
